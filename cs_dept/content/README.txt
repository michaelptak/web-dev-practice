https://placeholdr.ai/

To use it, you just provide a prompt and you'll be given a session ID. It will generate a few images in the backend linked to the session ID. You pop that session ID in this format (https://placeholdr.ai/{session_id}/{width}/{height}) and it'll select a random image from your session pool, resized to the dimensions you specified.
